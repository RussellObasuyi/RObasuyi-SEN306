# Software Construction

This repository contains materials, assignments, and projects for the Software Construction course.

# 📌 Overview

The course focuses on the principles and practices of building reliable, maintainable, and efficient software systems.

# 🛠️ Topics Covered
- Software design principles
- Object-oriented programming (OOP)
- Code structure and modularity
- Testing and debugging
- Version control
